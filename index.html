Vue Map

Vue 3 Components for your project using Leaflet map.

Get Started → Components

Easy to use

Most component is ready for use without specifying propetries.

Using DataSource

Сomponents use abstract data sources to connect any api.

Examples

Many examples will help you quickly understand how to use components.

# Installation

$ npm install vue-map-ui leaflet

$ yarn add vue-map-ui leaflet

$ pnpm install vue-map-ui leaflet

# Usage

<script setup>
import { VMap, VMapOsmTileLayer, VMapZoomControl } from 'vue-map-ui';
</script>

<template>
  <VMap style="height: 200px;">
    <VMapOsmTileLayer />
    <VMapZoomControl />
  </VMap>
</template>
